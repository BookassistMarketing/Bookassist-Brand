==========================================
QUICK TIPS
==========================================
- Type a slash command and press Enter to invoke a skill
- Skills guide you through a structured workflow step by step
- Plugins marked "MCP tools" give Claude extra abilities automatically
- To install new plugins: type /plugin inside Claude Code and search by name
- To reload after installing: type /reload-plugins
